This add-on enables you to shrink images in the editor/browser by default for easier navigation. In Anki 2.1.54, a new image resizer was introduced and you can double click images to shrink or enlarge them. How to use this add-on: In the Anki browser you can select notes, right click and choose the "Shrink Images" action to shrink all images in the selected notes. This action is undo-able (Edit -> Undo "Shrink Images"). When you paste an image in the note editor, it will be shrunk by default. You can disable this behavior in the add-on configuration (Tools->Add-ons-> click the add-on name -> click the "config" button) For both use cases, the images are only shrunk in the editor. They will have the same size as before when displayed on a card while reviewing.
